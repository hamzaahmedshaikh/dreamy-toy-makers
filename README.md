# Skylar 3D Toys

Custom 3D printed anime character toys. Upload your original character and get a unique collectible figure.

## Features

- **Image Upload** - Upload your OC image and preview your custom toy design
- **Payment Options** - PayPal and alternative payment methods available
- **Email Notifications** - Instant order confirmations and updates

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
